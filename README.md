# राजपूताना डिजिटल महल | Rajputana Digital Palace

> A culturally rich, immersive digital palace showcasing the complete heritage of Rajputana - the land of valor, honor, and cultural magnificence.

## 🏰 Project Overview

The Rajputana Digital Palace is a comprehensive web application that serves as a living archive and immersive storytelling hub for the rich history, culture, and heritage of Rajputana (modern-day Rajasthan). This digital palace allows users to explore the complete documented history of Rajput clans, dynasties, kings, warriors, forts, and cultural legacy through interactive storytelling, multimedia content, and authentic cultural design.

## ✨ Features

### 🎭 Cultural Design
- **Authentic Rajasthani Theme**: Sandstone, saffron, maroon, and gold color palette
- **Traditional Typography**: Cinzel, Playfair Display, and Cormorant Garamond fonts
- **Cultural Elements**: Palace arches, fresco textures, Rajasthani patterns
- **Festival Mode**: Dynamic UI transformations during cultural festivals
- **Responsive Design**: Optimized for all devices

### 📚 Content Sections

#### 👑 Dynasties (`/dynasties`)
- **Major Royal Houses**: Mewar, Marwar, Amber, Bikaner, Jaisalmer
- **Comprehensive Profiles**: Founders, notable rulers, achievements, and legacy
- **Search & Filter**: By name, region, or time period
- **Interactive Cards**: Detailed modal views with complete dynasty information

#### 🏰 Forts & Palaces (`/forts`)
- **Magnificent Fortresses**: Chittorgarh, Kumbhalgarh, Mehrangarh, Jaisalmer, Amber, Junagarh
- **Architectural Details**: Construction features, historical significance
- **UNESCO Heritage Sites**: Special highlighting of world heritage monuments
- **Virtual Tours**: Detailed descriptions and architectural features

#### 📖 Stories & Legends (`/stories`)
- **Heroic Sagas**: Maharana Pratap, Prithviraj Chauhan, Rani Padmini
- **Spiritual Journeys**: Mirabai's devotional path
- **Foundation Legends**: Bappa Rawal and dynasty origins
- **Audio Support**: Immersive storytelling with narration capabilities
- **Moral Lessons**: Educational content with historical context

### 🎨 Interactive Features
- **Daily Quotes**: Rotating inspirational quotes from Rajput tradition
- **Festival Mode Toggle**: Special UI effects and themes
- **Multi-language Support**: Hindi (primary), English, Rajasthani, Gujarati
- **Search Functionality**: Comprehensive search across all content
- **Responsive Navigation**: Seamless browsing experience

## 🛠️ Technology Stack

### Frontend
- **Framework**: Next.js 15.5.3 with TypeScript
- **Styling**: TailwindCSS v4 with custom Rajasthani theme
- **Fonts**: Google Fonts (Cinzel, Playfair Display, Cormorant Garamond)
- **Icons**: Unicode emojis and custom cultural symbols
- **Animations**: CSS transitions and hover effects

### Development Tools
- **Build Tool**: Turbopack (Next.js)
- **Package Manager**: npm
- **Linting**: ESLint with Next.js configuration
- **TypeScript**: Full type safety

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd rajputana-palace
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Open in browser**
   ```
   http://localhost:3000
   ```

### Build for Production

```bash
npm run build
npm start
```

## 🎨 Design System
